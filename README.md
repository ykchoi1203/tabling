## Spring-boot Store API     

### 프로젝트 설명
- 스프링부트를 이용하여 만든 Store API입니다.

### 개발 환경
- 운영체제 : Windows
- IDE : IntelliJ
- JAVA Version : JDK 17
- SpringBoot Version : 2.5.6
- 데이터 베이스 : MariaDB
- 빌드 툴 : Gradle
- 관리 툴 : GitHub

## Dependencies
- Spring Data JPA
- MariaDB connector
- Spring Security
- jwt
- swagger
- Lombok
- Spring Web


## 기술 스택
- 백엔드
    - SpringBoot, Spring Data JPA
- 데이터베이스
    - MariaDB
